Get started with XenDesign

To use XenDesign in your site simply follow the below installation procedure.

Just copy the below code and paste inside HEAD tag and then you are ready to rock with this framework

<link rel="stylesheet" href="https://xendesign.netlify.app/css/index.css">
<script src="https://xendesign.netlify.app/js/index.js" defer></script>
